# Wren's World Ideas

## Vibe

- Warm, cozy, colorful, and confidence-building.
- Short loops, big feedback, no pressure.
- Easy for a young kid to tap, hear, repeat, and celebrate.

## Possible Next Updates

- Add a simple "grown-up mode" toggle for editing settings without affecting play.
- Add a tiny parent note in each zone describing what the zone is practicing.
- Give Tee more moods and animations between rounds.
- Add a sticker book where earned stars unlock little scene decorations.
- Add a "Wren's Favorites" playlist inside Music Maker.
- Split the single HTML file into `src/` files once the project grows.

## Technical Notes

- The game currently saves progress in browser `localStorage` under `wrensWorld`.
- Speech uses the browser's built-in `speechSynthesis`.
- Sound effects use the browser's Web Audio API.
- No internet connection is required to play.
