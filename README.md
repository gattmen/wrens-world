# Wren's World

A small browser game for Wren, currently built as one self-contained HTML file.

## Open In VS Code

In VS Code: File -> Open Folder -> choose this `wrens-world` folder.

## Play The Game

Open this file directly in a browser:

```text
index.html
```

For a local server in VS Code:

1. Open the Command Palette.
2. Choose `Tasks: Run Task`.
3. Pick `Serve Wren's World`.
4. Visit `http://127.0.0.1:5177`.

## Editing Rhythm

1. Make one small change.
2. Refresh the browser.
3. Play the changed zone.
4. Add a note to `docs/ideas.md` if you spot the next idea.
5. Commit working versions with Git.
